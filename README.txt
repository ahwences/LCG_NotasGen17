Notas colaborativas de las clases de Estadística y R hechas por los alumnos. El sistema de control de versiones escogido es git, con acceso desde github, para que además los alumnos se entrenen en la escritura de código/documentos de manera colaborativa utilizando herramientas formales. La regla más importante es que cada vez que hagan un commit escriban una descripción acertada de los cambios que hicieron, y además que traten de modificar los archivos de poco en poco, haciendo los commits pertinentes para que el historial de cambios sea lo más claro (y al mismo tiempo conciso) posible. Los ayudantes y el profesor sólo intervendrán en el documento para verificar el contenido de las notas y de ser necesario señalar y/o realizar posibles correcciones.
